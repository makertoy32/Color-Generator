# Random Color Generator

A simple mini project built using HTML, CSS, and JavaScript that generates unique random RGB colors for boxes every time the button is clicked.

## Features

- Generates random RGB colors
- Each box gets a unique color
- Displays RGB values beside each box
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly DOM manipulation project

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How It Works

- JavaScript selects all boxes using `querySelectorAll()`
- Random RGB values are generated using `Math.random()`
- Colors are applied dynamically using `style.backgroundColor`
- RGB values are displayed using `textContent`
